ソースコードをハイライトしてHTMLに表示する

前準備

<script src="http://isystk.github.com/google-code-prettify/prettify.js" type="text/javascript">

function prettify() {
	prettyPrint();
}
if (window.addEventListener) {
	window.addEventListener("load", prettify, false);
} else if (window.attachEvent) {
	window.attachEvent("onload", prettify);
} else {
	window.onload = prettify;
}

ソースコードを表示するpreタグのスタイルを以下のように指定します。

<pre class="prettyprint linenums">
	ここにソースコードを記述します。
</pre>
